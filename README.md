# 🧑‍💼 Mini CRM - Client Lead Management System

A simple full-stack Mini CRM application to manage and track customer leads.

---

## 🚀 Features

- Add new customers (Name, Email, Phone)
- View all customer leads in a table
- Update customer details
- Delete customer records
- 🔥 Update Lead Status (New / Contacted)
- 🔍 Search customers by name
- Real-time UI update without page reload

---

## 🛠️ Tech Stack

**Frontend:**
- HTML
- CSS
- JavaScript

**Backend:**
- Node.js
- Express.js

**Database:**
- MongoDB (Mongoose)
